### D5 — Shared component: `<StudentComposerForm simulationLocked={bool} />`

File: `apps/web/src/components/student/StudentComposerForm.tsx` (new)

Single-page form with three section dividers (mirrors S4 teacher form):

1. **Bank Pick** — pathway selector (dropdown from `usePathways()`)
2. **Configure** — item count (5–80, integer), difficulty distribution (easy/mid/hard integer
   counts summing to item_count), time limit (300 000–10 800 000 ms selector),
   simulation toggle (hidden when `simulationLocked={true}`; always-on implicitly)
3. **Start** — submit button

Submit calls `useCreateSession` with:
```
{
  mode: 'exam',           // ALWAYS — §N Trap 2
  engine_type: 'linear',
  pathway_id: selected,
  composer_params: { item_count, difficulty_distribution, time_limit_ms },
  simulation_params: simulationLocked || toggleOn
    ? { no_back_nav: true, hide_feedback_until_submit: true }
    : undefined
}
```

On success: `router.push('/session/${data.session_id}/exam')`.

States within form: Loading (pathways fetch) / Empty (no pathways) / Error / 402-upgrade /
Content (form rendered).
